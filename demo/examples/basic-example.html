Spline 3D Parameter Debugging Tool Demo

Basic Example - Showing Core Features and Operations

Feature Overview

This example demonstrates the basic features of the Spline 3D Parameter Debugging Tool:

  • Camera Position Control: Use sliders to adjust XYZ axis positions
  • Background Color Control: Select colors from the palette or enter custom colors
  • Ground Control: Adjust the 3D scene ground material
  • API Information Viewer: Click the "Print API Info" button

How to Use

Try the following operations:

  1. Click the Basic Control tab to adjust the view ratio
  2. Switch to Camera Control and adjust the XYZ axis sliders
  3. Try applying preset camera positions
  4. Select different colors in the Background Control tab
